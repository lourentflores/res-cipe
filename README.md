# RES-CIPE 

## About the project
Res-cipe allows you to track your job applications. With the click of a button you can rate your applications by how excited you are to get the job and where you are in the application process. Res-cipe also tracks the success of each individual resum√© by providing metrics based on the results of your applications.

## Installation
1. Fork and clone the repo
2. Use the package manager [npm](https://docs.npmjs.com/cli/v6/commands/npm-install) to install Res-cipe.
```bash
npm install
```
